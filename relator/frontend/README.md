# Realtor.ai Frontend (Angular)

## Setup

```bash
cd frontend
npm install
ng serve
```

## Build

```bash
ng build
```

## Deploy to AWS S3/CloudFront

See `/aws-deploy/frontend-s3-deploy.sh` and `/aws-deploy/README.md`.

## Features

- Multistep forms for buyer info, requirements, finance, payment
- Speech-to-text with AI prefill
- Razorpay payment integration
